Notion 設計特色

Whisper Border
Notion 使用極細的邊框來定義空間，而非強烈的分隔線。透過控制 rgba 的 alpha 值，讓邊框若有似無，營造出乾淨卻有層次的介面。
opacity 0.05
最輕
rgba(0, 0, 0, 0.05)
幾乎隱形的分隔
opacity 0.10
標準
rgba(0, 0, 0, 0.1)
Notion 的主要選擇
opacity 0.20
明顯
rgba(0, 0, 0, 0.2)
需要更強調時使用
Warm vs Standard White
Notion 的背景色不是冷灰，而是帶一點黃調的暖白。這個微妙的差異讓整個介面感覺更溫暖、更人性化。
Notion Warm White
#f6f5f4
帶黃色底調的暖白，讓人聯想到紙張的質感，減少螢幕的冰冷感。
Notion 的選擇
Standard Neutral
#f8f8f8
業界常見的中性灰白，乾淨但缺少溫度。
Active State Animation
Notion 的按鈕在按下時會有一個快速的 scale(0.9) 縮放效果，讓使用者感受到明確的觸覺回饋。這個動畫快速而不誇張，符合 Notion 內斂的性格。
按下按鈕感受 scale(0.9) 效果
:active { transform: scale(0.9); } — transition: 0.08s ease
