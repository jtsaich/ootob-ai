WIRED 設計特色

WIRED 的設計以排版為結構、以留白為節奏。沒有圓角、沒有陰影，只用字型層級和髮線規則建立秩序。以下用這個頁面的內容，展示四個標誌性的設計模式。

Four Typeface Roles

WIRED 用四種字型各司其職：Display serif 抓住視線、Body serif 承載閱讀、UI sans 執行操作、Mono kicker 標記分類。同一句話，四種聲音。

Display 品味不可說 Playfair Display / 36px
Body 品味不可說 Lora / 18px
UI Sans 品味不可說 Inter 700 / 18px
Mono Kicker 品味不可說 JetBrains Mono / 15px

Editorial Grid

不靠卡片和陰影，只靠髮線規則和留白分隔內容。這是報刊排版的靈魂 — 讓文字自己建立結構。

Mono Kicker System

Square Everything

WIRED 的按鈕、卡片、輸入框一律 border-radius: 0。方正的邊緣傳達出技術感和印刷品質。對比一般 SaaS 常見的 8px 圓角，差異巨大。
