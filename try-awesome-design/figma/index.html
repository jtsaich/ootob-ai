Figma 設計特色

Figma 的設計語言以純粹的黑白介面搭配鮮豔的漸層色塊為標誌。以下用這個頁面的內容，展示三個標誌性的設計模式。

Monochrome + Gradient

Figma 的核心視覺張力 — 介面永遠是純黑白，色彩只出現在漸層 hero 區塊。兩者並置時，克制與活力的對比讓人一眼認出。

Interface Chrome
Black & White
#000000 和 #FFFFFF，沒有灰階捷徑。所有 UI 元素只用這兩色加上透明度。
Pill Button
Hero Section
Vibrant Color
Green, yellow, purple, pink — 色彩只在這裡爆發，讓漸層成為視覺焦點。
Glass Button

Variable Weight Scale

Figma 使用 Inter 的可變字重，從極輕的 300 到粗體 700，每一級都有精確的用途。以下展示五個關鍵字重。
